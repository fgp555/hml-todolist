About To-Do List App

The To-Do List app helps you stay organized and keep track of your daily tasks. You can add new tasks, mark them as complete, and delete them as you go. This app uses your browser's local storage to save your tasks, so you can access them even after refreshing the page.

Features:

  • Add new tasks to your list.
  • Mark tasks as completed by checking the checkbox.
  • Delete tasks when they are no longer needed.
  • Tasks are stored in your browser's local storage, so they persist between sessions.

How It Works:

The app relies on JavaScript to manage the tasks and local storage. When you add a task, it is saved to the local storage, and when you load the page again, the tasks are loaded from local storage. You can toggle completion status by clicking the checkbox next to each task, and delete tasks by clicking the "Delete" button next to them.

Technologies Used:

  • HTML5
  • CSS (with Pico CSS for styling)
  • JavaScript
  • LocalStorage API for saving tasks between sessions
